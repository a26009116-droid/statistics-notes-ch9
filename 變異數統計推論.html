11.1 χ² 分配 LO 11.1

χ²（卡方）分配用於對單一母體變異數 σ² 進行統計推論。

χ² 統計量

從常態母體隨機抽取 n 個樣本，樣本變異數 s² 滿足：

χ²df = (n−1)s² / σ²
df = n − 1

此統計量服從自由度為 df = n−1 的 χ² 分配。

11.2 單一母體變異數推論 LO 11.2

變異數的信賴區間

(1−α)×100% CI for σ²：

[ (n−1)s² / χ²α/2 , (n−1)s² / χ²1−α/2 ]

標準差 CI = 取兩端開根號

變異數的假設檢定

H₀: σ² = σ₀²

χ²df = (n−1)s² / σ₀²
df = n − 1

右尾：Hₐ: σ² > σ₀²
左尾：Hₐ: σ² < σ₀²
雙尾：Hₐ: σ² ≠ σ₀²

例題

例題 11.2｜成長基金 vs 價值基金 — 95% 標準差信賴區間 ▼

n = 36，成長基金 s = 23.0960%，價值基金 s = 17.9980%，計算 95% CI。

成長基金 σ² CI：

df=35，χ²0.025,35=53.203，χ²0.975,35=20.569

s²=(23.0960)²=533.43
下界=(35×533.43)/53.203=351.14
上界=(35×533.43)/20.569=907.98
σ CI=[√351.14, √907.98]=[18.74%, 30.13%]

價值基金 σ² CI：

s²=(17.9980)²=323.93
下界=(35×323.93)/53.203=213.17
上界=(35×323.93)/20.569=550.60
σ CI=[√213.17, √550.60]=[14.60%, 23.46%]

可見成長基金的不確定性（風險區間）比價值基金更寬。

例題 11.3｜成長基金標準差是否 > 20%？（右尾，α=0.01） ▼

H₀: σ²=400（σ=20%），Hₐ: σ²>400；n=36，s=23.0960

χ² = (35)(23.0960²)/400 = (35)(533.43)/400 = 46.67
p-value = P(χ²35 > 46.67) ≈ 0.085

α = 0.01 < p = 0.085
不拒絕 H₀

在 1% 顯著水準下，無足夠證據認為成長基金的風險（σ²）超過 400。

例題 11.4｜價值基金標準差是否 ≠ 15%？（雙尾，α=0.05） ▼

H₀: σ²=225（σ=15%），Hₐ: σ²≠225；n=36，s=17.9980

χ² = (35)(17.9980²)/225 = (35)(323.93)/225 = 50.39
因為 χ² > σ₀²，p-value = 2×P(χ²35 > 50.39) ≈ 0.091

α = 0.05 < p = 0.091
不拒絕 H₀

在 5% 顯著水準下，無法認為價值基金的標準差顯著不同於 15%。

11.3 F 分配 LO 11.3

F 分配用於比較兩個母體的變異數。

F 分配定義

當 σ₁²=σ₂² 時，來自兩常態母體的獨立樣本滿足：

Fdf₁,df₂ = s₁² / s₂²

df₁ = n₁ − 1，df₂ = n₂ − 1

F 分配由兩組自由度共同決定，正偏態，值域 [0, ∞)。

F 分配特性

0 ~ ∞

正偏
右偏（df 大時趨對稱）
df₁,df₂
兩組自由度
常態
兩母體需常態分配

11.4 兩母體變異數比推論 LO 11.4

σ₁²/σ₂² 的信賴區間

(1−α)×100% CI for σ₁²/σ₂²：

[ (s₁²/s₂²)×(1/Fα/2(df₁,df₂)) ,
(s₁²/s₂²)×Fα/2(df₂,df₁) ]

σ₁²/σ₂² 的假設檢定

H₀: σ₁²/σ₂² = 1

Fdf₁,df₂ = s₁² / s₂²

右尾：Hₐ: σ₁² > σ₂²
左尾：Hₐ: σ₁² < σ₂²
雙尾：Hₐ: σ₁² ≠ σ₂²

例題

例題 11.6｜統計考試兩班變異數比 95% CI ▼

n₁=11，s₁²=182.25；n₂=15，s₂²=192.25；95% CI for σ₁²/σ₂²

df₁=10，df₂=14，α/2=0.025
s₁²/s₂² = 182.25/192.25 = 0.948
F0.025(10,14) = 3.08，F0.025(14,10) = 3.70

下界 = 0.948 / 3.08 = 0.308
上界 = 0.948 × 3.70 = 1.507
95% CI = [0.31, 1.51]

CI 包含 1，不能認為兩班變異數有顯著差異。

例題 11.8｜成長基金風險 > 價值基金風險？（右尾，α=0.01） ▼

H₀: σ₁²/σ₂²=1，Hₐ: σ₁²/σ₂²>1（成長基金更險）

n₁=n₂=36，s₁=23.0960（成長），s₂=17.9980（價值）

F35,35 = (23.0960)²/(17.9980)² = 533.43/323.93 = 1.647
p-value = P(F35,35 > 1.647) ≈ 0.062

α=0.01 < p=0.062
不拒絕 H₀

在 1% 顯著水準下，無法證明成長基金比價值基金更具風險性。

例題 11.9｜汽車零件 A、B 的日需求變異數是否不同？（雙尾，α=0.05） ▼

H₀: σ₁²/σ₂²=1，Hₐ: σ₁²/σ₂²≠1；n₁=n₂=30

F29,29 = VAR_A / VAR_B = 1.105
p-value = 2×P(F29,29 > 1.105) ≈ 0.790

α=0.05 < p=0.790
不拒絕 H₀

在 5% 顯著水準下，無法認為兩零件的需求變異數有顯著差異。
