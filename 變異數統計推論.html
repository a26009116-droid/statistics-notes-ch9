11.1 χ² 分配的特性 LO 11.1

抽樣分配

若從常態母體抽取 n 筆觀測值，則：

χ²df = (n−1)S² / σ²

服從自由度 df = n−1 的 χ² 分配

S² 為樣本變異數，σ² 為母體變異數。此公式在母體為常態分配時才成立。

χ² 分配特性

0 ~ ∞
值域（永遠非負）
右偏
正偏態，df 越大越對稱
df
每個 df 對應一條曲線
右尾
χ²α,df 表右尾面積=α

查表符號說明

χ²α,df：右尾面積 = α，P(χ²df ≥ χ²α,df) = α
χ²1−α,df：左尾面積 = α，P(χ²df < χ²1−α,df) = α

Excel: =CHISQ.INV(cumulprob, df)　　→ 左尾
Excel: =CHISQ.INV.RT(α, df)　　　　→ 右尾
R: qchisq(cumulprob, df, lower.tail=TRUE)
例：χ²0.05,10 = 18.307（右尾5%）；χ²0.95,10 = 3.940（右尾95%，即左尾5%）
例題 11.1｜查找 χ² 臨界值▼

a. P(χ²₅ ≥ x) = 0.025，求 x

右尾面積 = 0.025 → 查 χ²0.025,5
Excel: =CHISQ.INV(0.975, 5) = 12.833
即 P(χ²₅ ≥ 12.833) = 0.025

b. P(χ²₈ < x) = 0.025，求 x

左尾面積 = 0.025 → 等同右尾面積 = 0.975 → 查 χ²0.975,8
Excel: =CHISQ.INV(0.025, 8) = 2.180
即 P(χ²₈ < 2.180) = 0.025

11.2 母體變異數的推論 LO 11.2

信賴區間 for σ²

[(n−1)s² / χ²α/2,df , (n−1)s² / χ²1−α/2,df]

df = n − 1

注意：χ² CI 不是對稱的「點估計 ± ME」格式！左右界用不同臨界值。

標準差 σ 的 CI：對上方各值取平方根即可。

假設檢定統計量 for σ²

χ²df = (n−1)s² / σ₀²

df = n − 1
σ₀² = H₀ 假設值
方向H₀Hₐp-value
右尾σ² ≤ σ₀²σ² > σ₀²P(χ²≥χ²stat)
左尾σ² ≥ σ₀²σ² < σ₀²P(χ²≤χ²stat)
雙尾σ² = σ₀²σ² ≠ σ₀²2×min(右尾,左尾)
雙尾 p-value 注意：若 s² > σ₀² → p = 2×P(χ²≥χ²stat)；若 s² < σ₀² → p = 2×P(χ²≤χ²stat)。

例題

例題 11.2｜成長基金與價值基金標準差的 95% CI（導言案例）▼
成長基金價值基金
s23.7993%17.9792%
n3636

成長基金（α=0.05，df=35）：

χ²0.025,35 = 53.203，χ²0.975,35 = 20.569

σ² CI: [(36−1)(23.7993)² / 53.203, (36−1)(23.7993)² / 20.569]
= [19872.59 / 53.203, 19872.59 / 20.569]
= [372.61, 963.77]

σ CI（開根號）= [19.30%, 31.04%]

價值基金：

σ CI = [14.58%, 23.45%]
95% 信心下，成長基金標準差介於 19.30%~31.04%
例題 11.3｜成長基金標準差是否 > 20%？（右尾，α=0.05）▼

等同檢定 σ² > 400（%²）。

H₀: σ² ≤ 400，Hₐ: σ² > 400（右尾）

n=36，s=23.7993，df=35

χ² = (36−1)(23.7993)² / 400
= 35 × 566.4109 / 400 = 49.561

p-value = P(χ²₃₅ ≥ 49.561) = 0.0524
不拒絕 H₀（p=0.0524 > 0.05）

在 5% 顯著水準下，無法確認成長基金標準差超過 20%。

例題 11.4｜價值基金標準差是否 ≠ 15%？（雙尾，α=0.05）▼

等同檢定 σ² ≠ 225（%²）。

H₀: σ² = 225，Hₐ: σ² ≠ 225（雙尾）

n=36，s=17.9792，df=35

χ² = (36−1)(17.9792)² / 225
= 35 × 323.2515 / 225 = 50.284

因 s²=323.25 > σ₀²=225，p-value = 2×P(χ²₃₅ ≥ 50.284) = 0.091
不拒絕 H₀（p=0.091 > 0.05）

在 5% 顯著水準下，無法認為價值基金標準差顯著異於 15%。

11.3 F 分配的特性 LO 11.3

F 分配定義

F(df₁,df₂) = (χ²df₁/df₁) / (χ²df₂/df₂)

當 σ₁² = σ₂² 時：
F(df₁,df₂) = S₁² / S₂²

df₁ = n₁−1（分子）
df₂ = n₂−1（分母）

F 分配特性

0 ~ ∞
值域（永遠非負）
右偏
df↑ 則趨近對稱
兩個
需要 df₁ 和 df₂
右尾
Fα,(df₁,df₂)

查表符號與左尾轉換

右尾：Fα,(df₁,df₂)，P(F ≥ Fα,(df₁,df₂)) = α

左尾轉換公式：
F1−α,(df₁,df₂) = 1 / Fα,(df₂,df₁)（注意 df 對調！）

Excel: =F.INV(cumulprob, df₁, df₂)
R: qf(cumulprob, df₁, df₂, lower.tail=TRUE)
例：F0.95,(6,8) = 1/F0.05,(8,6) = 1/4.15 = 0.24
例題 11.5｜查找 F 臨界值▼

a. P(F(7,10) ≥ x) = 0.025，求 x

F0.025,(7,10)：Excel =F.INV(0.975, 7, 10) = 3.95

b. P(F(7,10) < x) = 0.05，求 x

左尾面積=0.05 → F0.95,(7,10) = 1/F0.05,(10,7)
F0.05,(10,7) = 1/3.64 → F0.95,(7,10) = 0.27

11.4 兩母體變異數比的推論 LO 11.4

信賴區間 for σ₁²/σ₂²

[(s₁²/s₂²) × (1/Fα/2,(df₁,df₂)) ,
(s₁²/s₂²) × (1/F1−α/2,(df₁,df₂))]

等同於：
[(s₁²/s₂²)/Fα/2,(df₁,df₂) ,
(s₁²/s₂²)×Fα/2,(df₂,df₁)]

df₁=n₁−1，df₂=n₂−1

假設檢定統計量 for σ₁²/σ₂²

F(df₁,df₂) = s₁² / s₂²

df₁ = n₁−1，df₂ = n₂−1
方向H₀Hₐ
雙尾σ₁²/σ₂² = 1σ₁²/σ₂² ≠ 1
右尾σ₁²/σ₂² ≤ 1σ₁²/σ₂² > 1
左尾σ₁²/σ₂² ≥ 1σ₁²/σ₂² < 1
左尾檢定可將兩組對調為右尾，簡化計算。

例題

例題 11.6｜兩班統計考試分數變異數比的 95% CI▼

n1116

CI: [(182.25/457.96) × (1/3.06) , (182.25/457.96) × (1/0.284)]

成長基金（1）價值基金（2）
s23.7993%17.9792%

df3535

例題 11.9｜零件 A vs B 日需求變異數是否不同？（雙尾，α=0.05）▼

汽車零件分銷商對 A、B 兩種零件各記錄 30 天日需求量。

H₀: σ₁²/σ₂² = 1，Hₐ: σ₁²/σ₂² ≠ 1（雙尾）

Excel 輸出：F(29,29) = 1.105（s₁² 略大於 s₂²）
p-value = 2×P(F(29,29) ≥ 1.105) = 0.789
不拒絕 H₀（p=0.789 > 0.05）

在 5% 顯著水準下，兩種零件日需求量的變異數無顯著差異。

F 計算機（變異數比 σ₁²/σ₂²）

公式總整理

推論對象分配信賴區間檢定統計量df
σ²（單母體） χ²df [(n−1)s²/χ²α/2, (n−1)s²/χ²1−α/2] χ² = (n−1)s²/σ₀² n−1
σ（單母體） χ²df 對 σ² CI 各界取平方根 同 σ² 檢定 n−1
σ₁²/σ₂²（兩母體） F(df₁,df₂) [(s₁²/s₂²)/Fα/2,(df₁,df₂), (s₁²/s₂²)×Fα/2,(df₂,df₁)] F = s₁²/s₂² df₁=n₁−1, df₂=n₂−1

χ² vs F 分配比較

χ² 分配F 分配
用途單母體 σ²兩母體 σ₁²/σ₂²
參數1 個 df2 個 df（df₁, df₂）
值域[0, ∞)[0, ∞)
形狀正偏態正偏態
前提母體常態兩母體均常態

F 分配雙尾 p-value 計算

若 s₁² > s₂²（F > 1）：
p = 2 × P(F(df₁,df₂) ≥ Fstat)

若 s₁² < s₂²（F < 1）：
p = 2 × P(F(df₁,df₂) ≤ Fstat)

Excel: =2*F.DIST.RT(F_stat, df₁, df₂) ← s₁²>s₂²
R: 2*pf(F_stat, df₁, df₂, lower.tail=FALSE)

教授觀點｜考試重點分析

本章架構一張圖

變異數推論
├── 單母體 σ²（11.1~11.2）
│ ├── 分配：χ²df = (n−1)S²/σ²，df=n−1
│ ├── CI：[(n−1)s²/χ²α/2, (n−1)s²/χ²1−α/2]
│ └── 檢定：χ² = (n−1)s²/σ₀²
└── 兩母體 σ₁²/σ₂²（11.3~11.4）
├── 分配：F(df₁,df₂) = S₁²/S₂²
├── CI：用 Fα/2 和 F1−α/2（注意 df 對調）
└── 檢定：F = s₁²/s₂²

考試最常失分的地方

1. χ² CI 左右界搞反：下界用「大的 χ²（α/2）」，上界用「小的 χ²（1−α/2）」。分子(n−1)s²固定，除以越大的值結果越小。

2. F 左尾臨界值：F1−α,(df₁,df₂) = 1/Fα,(df₂,df₁)，df 必須對調！不對調就錯。

3. 雙尾 F 或 χ² 的 p-value：是 2×單尾面積，且要看 s² 與 σ₀² 的大小決定用哪個尾。

4. 前提條件：χ² 和 F 推論都要求母體服從常態分配，對非常態分配時需謹慎使用。

σ² 雙尾 p-value 判斷流程

計算 χ²stat = (n−1)s²/σ₀²

s² > σ₀²？
├── 是（χ²stat 偏大）→ p = 2 × P(χ² ≥ χ²stat)
└── 否（χ²stat 偏小）→ p = 2 × P(χ² ≤ χ²stat)
