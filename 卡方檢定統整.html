12.1 多項實驗的適合度檢定

多項實驗 (Multinomial Experiment)

由 n 個獨立且相同的試驗所組成，每次試驗有 k 種可能結果（類別），第 i 類的機率 pi 固定，且 p₁ + p₂ + ⋯ + pk = 1。

適合度檢定 (Goodness-of-Fit Test)

用來檢定母體各類別的比例是否等於某組假設值。

  • H₀：p₁ = p₁₀, p₂ = p₂₀, …, pk = pk0（各比例等於假設值）
  • Hₐ：並非所有比例都等於假設值
χ²df = Σ (oi − ei)² / ei
df = k − 1 ｜ ei = n × pi ｜ 每個 ei ≥ 5 時檢定有效 ｜ 右尾檢定
k = 類別數
oi = 觀測頻率
ei = 期望頻率 = n·pi
df = k−1
右尾檢定

前導範例：餐廳評分

某餐廳去年調查顧客評分比例：優秀15%、良好30%、普通45%、差10%。最近抽取250位顧客：優秀46、良好83、普通105、差16。α=5% 下，比例是否改變？

類別pi (H₀)觀測 oi期望 ei=250×pi(oi−ei)²/ei
優秀0.154637.51.9267
良好0.308375.00.8533
普通0.45105112.50.5000
差0.101625.03.2400
合計250χ²₃ = 6.520

p 值 = P(χ²₃ ≥ 6.520) = 0.089 > 0.05，不拒絕 H₀ — 無法得出比例已改變的結論。

12.2 獨立性卡方檢定

列聯表的獨立性檢定 (Test for Independence)

用來評估兩個類別變數是否有關聯。列聯表有 r 列、c 行。

H₀ : 兩個類別變數相互獨立
Hₐ : 兩個類別變數相依（有關聯）
期望頻率計算
eij = (第 i 列合計 × 第 j 行合計) / 總樣本數
χ²df = ΣΣ (oij − eij)² / eij
df = (r−1)(c−1) ｜ 每個 eij ≥ 5 時檢定有效 ｜ 右尾檢定

導入案例：健身中心入會決策 (Janet Mwangi)

從400位過去開放日參加者中，分析年齡組別與入會結果是否有關聯（α=5%）。

H₀ : 年齡組別與入會結果相互獨立
Hₐ : 年齡組別與入會結果相依
入會結果未滿30歲 (U)30~50歲 (B)超過50歲 (O)列合計
入會 (E)247244140
未入會 (N)848888260
行合計108160132400

期望頻率計算

入會結果UBO
入會 (E)(140×108)/400 = 37.80(140×160)/400 = 56.00(140×132)/400 = 46.20
未入會 (N)(260×108)/400 = 70.20(260×160)/400 = 104.00(260×132)/400 = 85.80

df = (2−1)(3−1) = 2

χ²₂ = (24−37.80)²/37.80 + (72−56.00)²/56.00 + (44−46.20)²/46.20 + (84−70.20)²/70.20 + (88−104.00)²/104.00 + (88−85.80)²/85.80

= 5.0349 + 4.5714 + 0.1048 + 2.7149 + 2.4615 + 0.0565 = 14.945

p 值 = P(χ²₂ ≥ 14.945) = 0.001 < 0.05，拒絕 H₀

在5%顯著水準下，年齡組別與入會結果有顯著關聯。45%的30~50歲組入會，22%的未滿30歲組入會，33%的超過50歲組入會。

12.3 常態性適合度檢定

卡方常態性檢定 (Chi-Square Test for Normality)

將數值型變數轉換為頻率分配（分成 k 個不重疊區間），然後計算在常態分配下各區間的期望頻率，再執行適合度檢定。

  • 需估計 2 個母數（μ 和 σ）→ 自由度減少 2
  • df = k − 3（k 個區間，估計 2 個母數）
χ²df = Σ (oi − ei)² / ei ，df = k − 3
ei = n × pi ，pi 由常態分配計算 ｜ 每個 ei ≥ 5 ｜ 右尾檢定
H₀ : 資料來自均值 μ̂、標準差 σ̂ 的常態分配
Hₐ : 資料不服從指定常態分配

範例 — 家戶所得常態性檢定 (Income)

+

一位經濟學家主張某中西部城市家戶年所得（千美元）不服從常態分配。以50筆資料（x̄=63.80, s=45.78）在5%顯著水準下進行檢定。

H₀ : 所得服從 N(63.80, 45.78²)
Hₐ : 所得不服從 N(63.80, 45.78²)

分成 k=5 個區間，在 H₀ 下計算各區間機率，ei = 50 × pi：

所得區間 (千$)觀測 oipi (常態)期望 ei(oi−ei)²/ei
< 2060.16938.4670.7190
20 ≤ 所得 < 40100.13226.6111.7368
40 ≤ 所得 < 60130.16538.2682.7091
60 ≤ 所得 < 80100.17148.5780.2394
≥ 80110.361718.0862.7762
合計 (n=50)50χ²₂ = 8.180

df = k−3 = 5−3 = 2

機率計算範例：P(X < 20) = P(Z < (20−63.80)/45.78) = P(Z < −0.9567) = 0.1693

p 值 = P(χ²₂ ≥ 8.180) = 0.017 < 0.05，拒絕 H₀

在5%顯著水準下，該城市家戶所得不服從均值63.80、標準差45.78的常態分配。

12.4 Jarque-Bera 常態性檢定

Jarque-Bera 檢定

常態性適合度檢定的替代方案，不需要將資料分組為區間，直接利用樣本的偏態係數 S 和（超額）峰態係數 K 進行檢定。

  • 偏態係數 S = 0：資料對稱於均值
  • 超額峰態係數 K = 0：分配與常態分配同高度（常態分配 S=K=0）
H₀ : S = 0 且 K = 0（資料來自常態分配）
Hₐ : S ≠ 0 或 K ≠ 0
χ²₂ = (n/6) [S² + K²/4]
df = 2 ｜ S = 偏態係數 ｜ K = 超額峰態係數 ｜ 右尾檢定

Excel 函數：SKEW（偏態）、KURT（超額峰態）；p 值：=CHISQ.DIST.RT(χ², 2)

範例 12.5 — JB 檢定家戶所得 (Income)

+

以1%顯著水準，對同一批所得資料（n=50）進行 Jarque-Bera 常態性檢定。

H₀ : S = 0 且 K = 0
Hₐ : S ≠ 0 或 K ≠ 0

由 Excel SKEW/KURT 函數計算得：S = 2.3190，K = 6.7322

χ²₂ = (50/6) [2.3190² + 6.7322²/4] = (8.333) [5.3778 + 11.3289] = (8.333)(16.7067) = 139.237

p 值 = P(χ²₂ ≥ 139.237) ≈ 0.000 < 0.01，拒絕 H₀

在1%顯著水準下，所得不服從常態分配——所得資料明顯右偏（S=2.319>0），與常態性適合度檢定結論一致。

案例研究：S&P 500 月報酬率

Javier Gonzalez 分析2012~2019年S&P 500指數的月報酬率（n=96），想確認報酬率是否服從常態分配，以做出適當投資推論。

統計量均值中位數標準差偏態 S峰態 K
值1.0376%1.6250%3.1736%−0.65251.0434

均值 (1.04%) < 中位數 (1.63%)，說明分配左偏（負偏態），符合 S = −0.6525 < 0 的觀察。

常態性適合度檢定 (GOF Test)

分成 k=5 個區間，由 N(1.0376, 3.1736²) 計算期望頻率：

df = 5−3 = 2，p 值 = P(χ²₂ ≥ 7.8412) = 0.0198 < 0.05，拒絕 H₀

Jarque-Bera 檢定

n=96, S=−0.6525, K=1.0434

χ²₂ = (96/6)[(−0.6525)² + (1.0434)²/4] = 16 × [0.4258 + 0.2722] = 16 × 0.6980 = 11.1665

p 值 = P(χ²₂ ≥ 11.1665) = 0.0038 < 0.05，拒絕 H₀

兩種檢定皆在5%顯著水準下拒絕常態性，S&P 500月報酬率不服從常態分配。

教授重點筆記

  • 卡方檢定統一公式：χ² = Σ(觀測−期望)²/期望，所有卡方檢定都套用這個邏輯。
  • 期望頻率條件：每個類別的期望頻率 ei 或 eij 必須 ≥ 5，否則需合併類別（如範例12.1合併廠商4+5）。
  • 適合度 vs 獨立性：前者只有1個類別變數（比較比例），後者有2個類別變數（看關聯性）。
  • 常態性 GOF 的 df = k−3，不是 k−1！因為從資料估計了 μ 和 σ 兩個母數。
  • JB 檢定永遠是 df=2，不隨樣本數或區間數改變。
  • JB 優點：不需分組、不受分組方式影響；缺點：與 R 的計算可能有微小差異（偏態/峰態係數的公式不同）。
  • 所有卡方類型的假設檢定都是右尾檢定（χ² 值只有正數，越大表示差異越大）。
  • 均值 < 中位數 → 左偏（負偏態，S < 0）；均值 > 中位數 → 右偏（正偏態，S > 0）。
