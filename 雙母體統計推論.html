10.1 兩母體平均數差的推論（獨立樣本）

點估計量：x̄₁ − x̄₂，用來估計母體差異 μ₁ − μ₂。

獨立樣本：兩樣本的抽取過程完全無關，互不影響。

三種情境的公式（依變異數已知程度區分）

情境①：σ₁²、σ₂² 已知 → 使用 z 分配
z = (x̄₁ − x̄₂ − d₀) / √(σ₁²/n₁ + σ₂²/n₂)
信賴區間：(x̄₁−x̄₂) ± zα/2 × √(σ₁²/n₁ + σ₂²/n₂)
母體標準差 σ 已知（較罕見，常來自歷史資料或大型研究）
情境②：σ 未知但假設相等 → 合併估計量 s²ₚ，t 分配，df = n₁+n₂−2
tdf = (x̄₁ − x̄₂ − d₀) / √(s²ₚ × (1/n₁ + 1/n₂))
s²ₚ = [(n₁−1)s₁² + (n₂−1)s₂²] / (n₁+n₂−2)　　df = n₁+n₂−2
信賴區間：(x̄₁−x̄₂) ± tα/2,df × √(s²ₚ(1/n₁+1/n₂))
合併估計量 s²ₚ 是兩樣本變異數的加權平均，以各自自由度為權重。
情境③：σ 未知且無法假設相等 → Welch t 分配，df 由公式決定
tdf = (x̄₁ − x̄₂ − d₀) / √(s₁²/n₁ + s₂²/n₂)
df = (s₁²/n₁ + s₂²/n₂)² / [(s₁²/n₁)²/(n₁−1) + (s₂²/n₂)²/(n₂−1)]（向下取整）
信賴區間：(x̄₁−x̄₂) ± tα/2,df × √(s₁²/n₁+s₂²/n₂)
df 通常為小數，取整後較保守。建議使用 Excel（t-Test: Unequal Variances）或 R 自動計算。

例題 10.1 — 香菸尼古丁含量（合併 t，95% CI）▼

比較 Brand A（n₁=20，x̄₁=1.68，s₁=0.22）與 Brand B（n₂=25，x̄₂=1.95，s₂=0.24）的尼古丁含量。σ 未知但假設相等。

s²ₚ = [(19)(0.22²)+(24)(0.24²)]/(43) = (0.9196+1.3824)/43 = 0.0535
df = 43，t0.025,43 = 2.017，SE = √(0.0535×(1/20+1/25)) = 0.0694
x̄₁−x̄₂ = 1.68−1.95 = −0.27
95% CI = −0.27 ± 2.017×0.0694 = [−0.41, −0.13]

0 不在 CI 內 → 兩品牌尼古丁含量有顯著差異（α=0.05）
例題 10.2 — 香菸尼古丁假設檢定（信賴區間法）▼

使用例 10.1 的 CI 進行雙尾假設檢定。

H₀: μ₁−μ₂ = 0Hₐ: μ₁−μ₂ ≠ 0
雙尾檢定（信賴區間法）

95% CI = [−0.41, −0.13]，不含 0
→ 拒絕 H₀

結論：兩品牌平均尼古丁含量有顯著差異。Brand B 顯著高於 Brand A。
例題 10.3 — 城市食品支出（σ 已知，右尾）▼

經濟學家聲稱城市1平均週食品支出 > 城市2。City 1：n₁=35，x̄₁=164，σ₁=12.50；City 2：n₂=30，x̄₂=159，σ₂=9.25。α=0.05

H₀: μ₁−μ₂ ≤ 0Hₐ: μ₁−μ₂ > 0
右尾檢定（σ 已知，z）

SE = √(12.50²/35 + 9.25²/30) = √(4.464+2.852) = 2.705
z = (164−159−0)/2.705 = 1.8485
p 值 = P(Z ≥ 1.8485) = 0.0323
0.0323 < 0.05 → 拒絕 H₀

結論：城市1平均食品支出顯著高於城市2。
例題 10.4 — 黃金 vs 石油年報酬率（Welch t，雙尾）▼

比較黃金與石油產業年均報酬率是否不同。各抽樣 n₁=n₂=10 家公司，σ 未知且不可假設相等。

黃金：x̄₁=17.2，Var₁=70.67；石油：x̄₂=18.0，Var₂=74.23

H₀: μ₁−μ₂ = 0Hₐ: μ₁−μ₂ ≠ 0
雙尾（Welch t）

Excel 計算：t = −0.3023，df = 17（Welch 公式），p 值 = 0.766
0.766 > 0.05 → 不拒絕 H₀

結論：兩產業年均報酬率無顯著差異（α=0.05）。

10.2 配對樣本均差的推論（μ_D）

配對樣本：對同一對象在兩個條件下各測量一次，差值 D = X₁ − X₂ 消除了個體間變異，通常比獨立樣本更有效率。

兩種配對方式

  • ① 同一對象前後測量（如：處理前/後）
  • ② 配對觀測值（如：雙胞胎、相似地點）

與獨立樣本的差異

  • 獨立樣本：兩組完全分開抽取
  • 配對樣本：每對觀測值相關聯
  • 配對可消除干擾變異，增加檢定力
配對樣本：估計量 D̄ 與檢定統計量
D̄ = (ΣD_i) / n　，　tdf = (D̄ − μ_D0) / (s_D / √n)
df = n − 1　｜　μ_D0：假設差值（通常為 0）
信賴區間：D̄ ± tα/2, df × s_D / √n

例題 10.5 — 工作效率前後測（配對 CI）▼

管理者測量 10 名員工在變更工作區域前後完成任務的時間（分鐘）。D = 前−後（差值越大表示效率提升越多）。

D̄ = 4.90，s_D = 9.78，n = 10，df = 9，t0.025,9 = 2.262
95% CI = 4.90 ± 2.262 × (9.78/√10) = 4.90 ± 7.00 = [−2.10, 11.90]

CI 包含 0 → 無法在 95% 信心水準下確認工作區域變更提升了效率。
例題 10.6 — 工作效率假設檢定（信賴區間法）▼
H₀: μ_D = 0（無改變）Hₐ: μ_D ≠ 0（有改變）
雙尾（配對 t）

95% CI = [−2.10, 11.90]，包含 0
→ 不拒絕 H₀

結論：在 5% 顯著水準下，無法確認工作區域配置的改變對效率有影響。
例題 10.7 — 卡路里標示對消費行為的影響（配對 t，右尾）▼

營養師針對 40 名顧客，記錄卡路里標示前後的飲料與食物熱量（配對設計：同一顧客前後比較）。

① 飲料熱量（D = 標示前−標示後）：

H₀: μ_D ≤ 0Hₐ: μ_D > 0
右尾

df = 39，t = 1.6286，p 值（單尾）= 0.0557
0.0557 > 0.05 → 不拒絕 H₀（飲料熱量下降未達顯著）

② 食物熱量：

t = 6.7795，p 值 ≈ 0.000
→ 拒絕 H₀（食物熱量顯著下降）

結論：卡路里標示對食物消費有顯著影響，但對飲料消費影響未達統計顯著。

10.3 兩母體比例差的推論（p₁ − p₂）

點估計量：p̂₁ − p̂₂，其中 p̂₁ = x₁/n₁，p̂₂ = x₂/n₂。

信賴區間
(p̂₁ − p̂₂) ± zα/2 × √(p̂₁(1−p̂₁)/n₁ + p̂₂(1−p̂₂)/n₂)
條件：n₁p̂₁, n₁(1−p̂₁), n₂p̂₂, n₂(1−p̂₂) 均 ≥ 5
假設檢定統計量
d₀=0：z = (p̂₁−p̂₂) / √(p̄(1−p̄)(1/n₁+1/n₂))，p̄ = (x₁+x₂)/(n₁+n₂)
d₀≠0：z = [(p̂₁−p̂₂) − d₀] / √(p̂₁(1−p̂₁)/n₁ + p̂₂(1−p̂₂)/n₂)

例題 10.8 — 候選人支持率 CI（兩比例差信賴區間）▼

三個月前：120 名登記選民中 51 人支持候選人 A（p̂₁=51/120=0.425）
現在：80 名登記選民中 44 人支持候選人 A（p̂₂=44/80=0.55）

p̂₁−p̂₂ = 0.425−0.55 = −0.125
95% CI：使用 z0.025=1.96，計算 SE = √(p̂₁(1−p̂₁)/120 + p̂₂(1−p̂₂)/80)
95% CI = [−0.0071, 0.1043]（含 0，即 −0.71% ~ 10.43%）

CI 包含 0 → 無法確認三個月來支持率有顯著變化。
例題 10.9 — 候選人支持率假設檢定▼
H₀: p₁−p₂ = 0Hₐ: p₁−p₂ ≠ 0
雙尾

CI [−0.0071, 0.1043] 包含 d₀ = 0
→ 不拒絕 H₀（支持率未有顯著改變）

例題 10.10 — 男女網路購物比例（右尾，d₀=0）▼

分析師聲稱男性定期網路購物比例 > 女性。
男性：n₁=6,000，x₁=5,400（p̂₁=0.90）；女性：n₂=10,000，x₂=8,000（p̂₂=0.80）

H₀: p₁−p₂ ≤ 0Hₐ: p₁−p₂ > 0
右尾（d₀=0，使用合併 p̄）

p̄ = (5400+8000)/(6000+10000) = 13400/16000 = 0.8375
z = (0.90−0.80)/√(0.8375×0.1625×(1/6000+1/10000)) = 7.4000
p 值 ≈ 0.000 → 拒絕 H₀

結論：男性網路購物比例顯著高於女性（α=0.05）。
例題 10.11 — 酒品盲測偏好（右尾，d₀=0.20）▼

同款酒，告知一組是 $25 高價酒（n₁=60，48 人喜歡，p̂₁=0.80），另一組是 $10 低價酒（n₂=50，20 人喜歡，p̂₂=0.40）。
欲確立第一組喜歡比例超過第二組 20 個百分點以上。

H₀: p₁−p₂ ≤ 0.20Hₐ: p₁−p₂ > 0.20
右尾（d₀=0.20，σ 未知用 p̂ 估計）

z = [(0.80−0.40)−0.20] / √(0.80×0.20/60 + 0.40×0.60/50)
= 0.20 / √(0.002667+0.0048) = 0.20/0.08638 = 2.3149
p 值 = P(Z ≥ 2.3149) = 0.0103
0.0103 < 0.05 → 拒絕 H₀

結論：高價標示組喜歡比例確實比低價標示組高出 20% 以上（α=0.05）。
此結果顯示「建議效果」對口味評斷有顯著影響。

研究者 Anka 欲確認：
① 女性想要文化背景相同的伴侶的比例，是否顯著高於男性
