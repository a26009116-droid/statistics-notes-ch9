10.1 兩母體均值差的推論（獨立樣本） LO 10.1

兩組完全獨立的隨機樣本，推論 μ₁ − μ₂。

信賴區間（三種情況）

情況 1：σ₁², σ₂² 已知
(x̄₁−x̄₂) ± zα/2 √(σ₁²/n₁ + σ₂²/n₂)
情況 2：σ₁²=σ₂²（未知但假設相等）
(x̄₁−x̄₂) ± tα/2,df √(sp²(1/n₁+1/n₂))

sp² = [(n₁−1)s₁² + (n₂−1)s₂²] / (n₁+n₂−2)
df = n₁ + n₂ − 2
情況 3：σ₁²≠σ₂²（未知且不相等）
(x̄₁−x̄₂) ± tα/2,df √(s₁²/n₁ + s₂²/n₂)

df = (s₁²/n₁ + s₂²/n₂)² / [(s₁²/n₁)²/(n₁−1) + (s₂²/n₂)²/(n₂−1)]（取整數下捨）

假設檢定統計量（三種情況）

情況條件統計量分配
1σ₁², σ₂² 已知z = (x̄₁−x̄₂−d₀) / √(σ₁²/n₁+σ₂²/n₂)Z
2σ 未知，相等t = (x̄₁−x̄₂−d₀) / √(sp²(1/n₁+1/n₂))tn₁+n₂−2
3σ 未知，不等t = (x̄₁−x̄₂−d₀) / √(s₁²/n₁+s₂²/n₂)tdf（Welch）

H₀ 設定：μ₁−μ₂＝d₀ / ≤d₀ / ≥d₀，d₀ 通常為 0。

均值差計算機（獨立樣本）

10.2 均值差的推論（配對樣本） LO 10.2

觀測值成對（前後測、兩兩配對），計算每對差值 d = x₁ − x₂，再用單樣本 t 推論。

核心概念

1
計算每對差值：dᵢ = x₁ᵢ − x₂ᵢ
2
計算 d̄（差值平均）與 sD（差值標準差）
3
使用 t 分配，df = n − 1
4
適用條件：D（≈）服從常態分配

公式

信賴區間：
d̄ ± tα/2, n−1 × (sD / √n)

檢定統計量：
t = (d̄ − d₀) / (sD / √n)

df = n − 1

配對樣本計算機

10.3 兩母體比例差的推論（獨立樣本） LO 10.3

推論 p₁ − p₂，使用 z 分配（大樣本）。適用條件：n₁p̄₁, n₁(1−p̄₁), n₂p̄₂, n₂(1−p̄₂) 均 ≥ 5。

信賴區間

(p̄₁−p̄₂) ± zα/2 √(p̄₁(1−p̄₁)/n₁ + p̄₂(1−p̄₂)/n₂)

p̄₁ = x₁/n₁，p̄₂ = x₂/n₂（樣本比例）

假設檢定統計量

d₀ = 0（合併估計量）：
z = (p̄₁−p̄₂) / √(p̄(1−p̄)(1/n₁+1/n₂))
p̄ = (x₁+x₂)/(n₁+n₂)

d₀ ≠ 0：
z = (p̄₁−p̄₂−d₀) / √(p̄₁(1−p̄₁)/n₁+p̄₂(1−p̄₂)/n₂)

例題

例題 10.10｜男性 vs 女性網購比例（右尾，d₀=0，α=0.05）▼

分析師宣稱男性網購比例高於女性。男性 n₁=6000，x₁=5400；女性 n₂=10000，x₂=8600。

H₀: p₁−p₂ ≤ 0，Hₐ: p₁−p₂ > 0（右尾）

p̄₁ = 0.90，p̄₂ = 0.86
p̄ = (5400+8600)/(6000+10000) = 0.875

z = (0.90−0.86) / √(0.875×0.125×(1/6000+1/10000))
= 0.04 / 0.005396 = 7.4066

p-value = P(Z ≥ 7.4066) ≈ 6.48 × 10⁻¹⁴
拒絕 H₀ — 男性網購比例顯著高於女性
例題 10.11｜盲測品酒：組1比例是否高於組2超過20個百分點？（右尾，d₀=0.20）▼

組1（60人，告知$25）：48人喜歡，p̄₁=0.80；組2（50人，告知$10）：20人喜歡，p̄₂=0.40。

H₀: p₁−p₂ ≤ 0.20，Hₐ: p₁−p₂ > 0.20（右尾，d₀=0.20）

z = (0.80−0.40−0.20) / √(0.80×0.20/60 + 0.40×0.60/50)
= 0.20 / 0.08485 = 2.3146

p-value = P(Z ≥ 2.3146) = 0.0103
拒絕 H₀（p=0.0103 < 0.05）

組1喜歡比例顯著高於組2超過20個百分點，暗示價格標籤對感知品質有影響。

教授觀點｜考試重點分析

本章核心邏輯

第10章是第9章的延伸：從「一個母體」擴展到「兩個母體的比較」。

雙母體推論
├── 均值差 μ₁−μ₂
│ ├── 獨立樣本（10.1）
│ │ ├── σ已知 → z統計量
│ │ ├── σ未知相等 → 合併t，df=n₁+n₂−2
│ │ └── σ未知不等 → Welch t，df用公式
│ └── 配對樣本（10.2）
│ └── 計算d=x₁−x₂ → 單樣本t，df=n−1
└── 比例差 p₁−p₂（10.3）
├── CI：用各自p̄計算SE
└── 假設檢定：d₀=0用合併p̄；d₀≠0用各自p̄

考試最常失分的地方

1. 獨立 vs 配對搞混：看到「before/after」「前後測」「相同對象」→ 配對；兩組來源不同 → 獨立。

2. σ 相等與不相等的公式：相等用合併sp²（df=n₁+n₂−2）；不相等用Welch df（題目通常會說或給軟體輸出）。

3. 比例差檢定的SE公式：d₀=0 → 用合併p̄；d₀≠0 → 用各自p̄。CI一律用各自p̄。

4. CI法做假設檢定（雙尾）：如果CI包含d₀（通常是0）→ 不拒絕H₀；不含d₀ → 拒絕H₀。只適用雙尾。

三種均值差情況選擇流程

題目給的是 σ₁, σ₂？
├── 是 → 情況1：z統計量
└── 否（給s₁, s₂）
├── 題目說「assume equal variance」→ 情況2：合併t
└── 題目說「not equal」或給軟體輸出 → 情況3：Welch t
