第1章 簡介

什麼是時間數列？

觀測值隨時間產生變化，彼此之間可能具有相關性。例如：每日氣溫、月銷售量、消費者物價指數（CPI）、犯罪率等。

時間數列分析的目標：利用資料間的時間相關性，建立合適模式，進行預測。

常用方法

  • 指數平滑法 (Exponential Smoothing)：以特定期數的移動平均做預測。
  • 分解法 (Decomposition)：利用趨勢(T)、季節(S)、循環(C)、隨機(I)建立模式。
  • ARIMA 模式：由 Box-Jenkins 提出，本課程主軸。

資料型態

  • 離散型：如每月盈收，在時間點 t₀, t₁, t₂, … 觀測得 Z₁, Z₂, Z₃, …
  • 連續型：如每日氣溫，利用等時間抽樣轉換為離散型。
第2章 平穩型隨機過程

平穩型時間數列 (Stationary Time Series)

一時間數列其統計特性不會隨時間變動而改變。

1. 嚴密的平穩型 (Strict Stationarity)

所有聯合機率分配皆不隨時間平移而改變。

2. 弱的平穩型 (Weak / Covariance Stationarity)

只有 1 階和 2 階動差不隨時間變動。實務上常用此定義。

  • 對所有 t，E(Zt) = μ（均值為常數）
  • 對所有 t，Var(Zt) = γ₀（變異數為常數）
  • Cov(Zt, Zt+k) = γk（只與間隔 k 有關，與 t 無關）

自我相關函數 (ACF)

自我互變異數 (Autocovariance)：γk = Cov(Zt, Zt+k)
估計式：γ̂k = (1/n) Σ (Zt−Z̄)(Zt+k−Z̄)
自我相關係數 (ACF)：ρk = γk / γ₀
估計式：rk = γ̂k / γ̂₀ ｜ 繪圖得自我相關函數圖 (ACF plot)
ρk 的顯著性檢定

H₀: ρk = 0（無自我相關）

當 n 很大時，rk ~ N(0, 1/n)，檢定統計量：

若 |rk| > 1.96/√n，則 ρk 顯著（α=5%）

範例：時間數列資料的 ACF 計算

+

資料：101, 103, 100, 98, 97, 100, 102, 99，n=8，Z̄ ≈ 100

計算 r₁, r₂, r₃：

krk1.96/√8判斷
1−0.31±0.693不顯著
2−0.05±0.693不顯著
3−0.08±0.693不顯著

因所有 |rk| < 0.693，故資料無顯著自我相關，接近白干擾。

偏自我相關函數 (PACF)

考慮 Zt 與 Zt-k，在移除中間期 Zt-1, …, Zt-k+1 的影響後，兩者的淨相關。

利用 Yule-Walker 方程式計算 φkk（PACF 在 k 階的估計值）
繪圖得偏自我相關函數圖 (PACF plot)｜顯著標準同 ACF：|φ̂kk| > 1.96/√n
Yule-Walker 方程式（以 p 階 AR 為例）

ρ₁ = φ₁ + φ₂ρ₁ + … + φpρp-1
ρ₂ = φ₁ρ₁ + φ₂ + … + φpρp-2
…
ρp = φ₁ρp-1 + … + φp

用 Cramer 法則求解 φkk。

第3章 平穩型時間數列模式

白干擾過程 (White Noise)

白干擾 at 的定義
  • E(at) = 0，對所有 t
  • Var(at) = σa²，對所有 t（常數）
  • Cov(at, as) = 0，對所有 t ≠ s（互不相關）

後移運算子 (Backward Shift Operator)：B Zt = Zt-1，B² Zt = Zt-2，依此類推。

自我迴歸過程 AR(p)

Zt = φ₁Zt-1 + φ₂Zt-2 + … + φpZt-p + at
後移運算子形式：φ(B) Zt = at，其中 φ(B) = 1 − φ₁B − φ₂B² − … − φpBp

平穩型條件：φ(B)=0 的根落在單位圓之外（|根| > 1）。

AR(1)

Zt = φ₁Zt-1 + at，平穩條件：|φ₁| < 1

ACF：逐漸遞減（幾何或震盪）
PACF：1 期後切斷（φ̂₁₁≠0, φ̂₂₂≈0）
AR(2)

Zt = φ₁Zt-1 + φ₂Zt-2 + at

ACF：逐漸遞減
PACF：2 期後切斷
AR(p)

一般 p 階

ACF：逐漸遞減
PACF：p 期後切斷

AR(1) 詳細公式

+

模式：Zt = φ₁Zt-1 + at

變異數：γ₀ = σa² / (1 − φ₁²)

自我互變異數：γk = φ₁k × γ₀

ACF：ρk = φ₁k（幾何遞減，若 φ₁>0 同號，若 φ₁<0 交替震盪）

PACF：φ₁₁ = ρ₁ = φ₁，φ₂₂ = φ₃₃ = … = 0 → 1 期後切斷

φ₁平穩？ACF ρk 形狀
0.8是（|0.8|<1）正值緩慢遞減：0.8, 0.64, 0.51, …
−0.6是（|−0.6|<1）交替震盪：−0.6, 0.36, −0.22, …
1.2否（|1.2|>1）爆炸性增長，非平穩

AR(2) 詳細公式

+

模式：Zt = φ₁Zt-1 + φ₂Zt-2 + at

平穩條件：φ(B)=1−φ₁B−φ₂B²=0 的根落在單位圓外，等效於：φ₁+φ₂<1, φ₂−φ₁<1, |φ₂|<1

Yule-Walker 方程式：

ρ₁ = φ₁/(1−φ₂)，ρ₂ = φ₁ρ₁ + φ₂

ρk = φ₁ρk-1 + φ₂ρk-2（k ≥ 2，逐漸遞減）

PACF：φ₁₁=ρ₁，φ₂₂=φ₂ ≠ 0，φ₃₃=… = 0 → 2 期後切斷

移動平均過程 MA(q)

Zt = at − θ₁at-1 − θ₂at-2 − … − θqat-q
後移運算子形式：Zt = θ(B) at，其中 θ(B) = 1 − θ₁B − θ₂B² − … − θqBq

可逆性條件 (Invertibility)：θ(B)=0 的根落在單位圓之外。

MA(1)

Zt = at − θ₁at-1

ACF：1 期後切斷（ρ₁≠0, ρ₂=ρ₃=…=0）
PACF：逐漸遞減
MA(2)

Zt = at − θ₁at-1 − θ₂at-2

ACF：2 期後切斷
PACF：逐漸遞減
MA(q)

一般 q 階

ACF：q 期後切斷
PACF：逐漸遞減

MA(1) 詳細公式

+

模式：Zt = at − θ₁at-1

變異數：γ₀ = (1+θ₁²)σa²

自我互變異數：γ₁ = −θ₁σa²，γk=0 for k≥2

ACF：ρ₁ = −θ₁/(1+θ₁²)，ρk=0 for k≥2 → 1 期後切斷

PACF：φkk = −(−θ₁)k(1−θ₁²2k)/(1−θ₁2k+2) → 逐漸遞減

自我迴歸移動平均過程 ARMA(p,q)

φ(B) Zt = θ(B) at
Zt = φ₁Zt-1+…+φpZt-p + at − θ₁at-1−…−θqat-q
  • 平穩條件：φ(B)=0 的根落在單位圓外
  • 可逆性條件：θ(B)=0 的根落在單位圓外
  • ACF：逐漸遞減（q 期後以 AR 型態遞減）
  • PACF：逐漸遞減（p 期後以 MA 型態遞減）

ARMA(1,1) 詳細公式

+

模式：Zt = φ₁Zt-1 + at − θ₁at-1

變異數：γ₀ = σa² (1−2φ₁θ₁+θ₁²) / (1−φ₁²)

ACF：ρ₁ = (1−φ₁θ₁)(φ₁−θ₁)/(1−2φ₁θ₁+θ₁²)

ρk = φ₁ ρk-1，for k ≥ 2 → 1 期後以 AR(1) 型態遞減

PACF：逐漸遞減

第4章 非平穩型時間數列 ARIMA

非平穩型的特性與差分

非平穩型：隨著時間變動，均值會不同（趨勢）。

解決方法：做差分 (Differencing)，轉換為平穩型。

差分運算子：∇ Zt = Zt − Zt-1 = (1−B) Zt
1 階差分：∇ Zt = (1−B) Zt
2 階差分：∇² Zt = (1−B)² Zt
d 階差分：∇d Zt = (1−B)d Zt
ARIMA(p, d, q) 過程

對 Zt 做 d 階差分，令 Wt = ∇d Zt，Wt 服從 ARMA(p,q) 模式：

φ(B)(1−B)d Zt = θ(B) at

p = AR 階數，d = 差分次數，q = MA 階數

常見 ARIMA 模式範例

模式方程式說明
ARIMA(0,1,0)Zt = Zt-1 + at隨機步 (Random Walk)
ARIMA(1,1,0)∇Zt = φ₁∇Zt-1 + at差分後 AR(1)
ARIMA(0,1,1)∇Zt = at − θ₁at-1差分後 MA(1)，常見模式
ARIMA(1,1,1)∇Zt = φ₁∇Zt-1 + at − θ₁at-1差分後 ARMA(1,1)

有常數項的 ARIMA

若 d=1 且有常數 δ，則 E(∇Zt) = μ ≠ 0，表示 Zt 有線性趨勢。

第5章 模式建立（識別）

Box-Jenkins 三步驟

  1. 識別 (Identification)：利用樣本 ACF 和 PACF 判斷 p、d、q 值。
  2. 估計 (Estimation)：用條件式或精確最小平方法估計參數。
  3. 診斷 (Diagnostic Checking)：檢驗殘差是否為白干擾。

模式識別步驟

  1. 看樣本 ACF：若緩慢遞減（或不收斂）→ 非平穩，需做差分。
  2. 做 d 次差分直到 ACF 快速遞減為止 → 確定 d 值。
  3. 由差分後的樣本 ACF 和 PACF 確定 p、q 值（見下表）。

識別規則

AR(p)
ACF：逐漸遞減
PACF：p 期後切斷 ✓

→ 取 q=0，由 PACF 切斷點定 p

MA(q)
ACF：q 期後切斷 ✓
PACF：逐漸遞減

→ 取 p=0，由 ACF 切斷點定 q

ARMA(p,q)
ACF：逐漸遞減
PACF：逐漸遞減

→ 兩者皆遞減，需試多個組合

White Noise
ACF：全不顯著
PACF：全不顯著

→ p=q=0，資料本身無相關

SAS 程式範例：ARIMA(0,1,1) 識別流程

+

以 n=197 筆資料（例如溫度序列）為例：

proc arima;
  identify var = z;         /* 查看原始資料 ACF/PACF */
proc arima;
  identify var = z(1);      /* 對 z 做 1 階差分，查看 ACF/PACF */
  estimate q=(1) printall plot;   /* 估計 MA(1) 參數 */
  forecast lead = 12;       /* 預測未來 12 期 */
run;

識別結果：原始資料 ACF 緩慢遞減 → d=1；差分後 ACF 在 lag=1 切斷 → q=1；PACF 逐漸遞減 → p=0。

∴ 模式為 ARIMA(0,1,1)：∇Zt = at − θ₁at-1

第6章 參數估計與模式診斷

參數估計

條件式最小平方法 (Conditional Least Squares)

對 MA 模式，概似函數含遺失值，設遺失值為 0（即 a0=a-1=…=0），得條件式概似函數。

非線性估計（MA/ARMA 模式）

MA 參數 θ 使 SSE(θ) 為非線性函數，使用Newton-Raphson 法（泰勒展開迭代）：

  1. 給定初始值 θ(0)
  2. 以 θ(0) 為中心做泰勒展開至一次項，轉換為線性問題
  3. 用最小平方求 Δθ，更新 θ(1) = θ(0) + Δθ
  4. 重複直到收斂（参數變動量 < 容忍值）

模式診斷

若模式合適，殘差 ât 應為白干擾（無自我相關）。

Ljung-Box 檢定（殘差白干擾性）

H₀：殘差無自我相關（模式合適）

Q = n(n+2) Σ [r̂k² / (n−k)] (Ljung-Box 統計量)
亦可用 Box-Pierce：Q = n Σ r̂k²｜服從 χ²(K−p−q)
  • 不拒絕 H₀ → 殘差為白干擾 → 模式合適
  • 拒絕 H₀ → 殘差有相關 → 模式需修正

最佳模式選擇準則

準則公式特性
AIC−2ln(L) + 2(p+q)有效性強，但不具一致性
SBC/BIC−2ln(L) + (p+q)ln(n)一致性強，但較保守
σ̂a²SSE / (n−p−q)白干擾變異數估計值，越小越好

比較多個候選模式時，以 AIC 或 SBC 最小者為優先。

第7章 預測

最佳預測值

設最後觀測值為 Zn，預測未來 l 期 Zn+l 的最佳預測值 Ẑn(l) 使 MSE 最小。

最佳預測值：Ẑn(l) = E(Zn+l | Zn, Zn-1, …)
條件期望值規則：
E(Zn+j | 已知) = Ẑn(j)，若 j > 0
E(Zn+j | 已知) = Zn+j，若 j ≤ 0
E(an+j | 已知) = 0，若 j > 0

各模式的預測公式

模式預測公式
AR(1)Ẑn(l) = φ₁ Ẑn(l−1)，Ẑn(1) = φ₁Zn
MA(1)Ẑn(1) = −θ₁ân，Ẑn(l) = 0 for l ≥ 2
ARIMA(0,1,1)Ẑn(l) = Zn − θ₁ân（常數）
ARMA(1,1)Ẑn(1) = φ₁Zn − θ₁ân；Ẑn(l) = φ₁Ẑn(l−1)

預測誤差與預測區間

en(l) = Zn+l − Ẑn(l) = an+l + ψ₁an+l-1 + … + ψl-1an+1
Var[en(l)] = σa²(1 + ψ₁² + … + ψl-1²)
95% 預測區間：Ẑn(l) ± 1.96 × √Var[en(l)]｜隨 l 增大，區間變寬

預測值的更新（修正）

當過了一期，得到新觀測值 Zn+1，可更新預測：

Ẑn+1(l−1) = Ẑn(l) + ψl-1 × ân+1
ân+1 = Zn+1 − Ẑn(1) （實際值 − 預測值）

第8章 季節性時間數列

季節性模式概述

許多時間數列具有季節性，如月份銷售（s=12）、季度資料（s=4）。

季節性 AR / MA 模式

模式方程式ACFPACF
SAR(1)sZt = Φ₁Zt-s + at在 s, 2s, 3s, … 逐漸遞減在 s 切斷
SAR(2)sZt = Φ₁Zt-s + Φ₂Zt-2s + at在 s, 2s, … 逐漸遞減在 2s 切斷
SMA(1)sZt = at − Θ₁at-s在 s 切斷在 s, 2s, … 逐漸遞減
SMA(2)sZt = at − Θ₁at-s − Θ₂at-2s在 2s 切斷在 s, 2s, … 逐漸遞減

相乘型季節性 ARIMA (Multiplicative SARIMA)

ARIMA(p,d,q)(P,D,Q)s
φ(B) Φ(Bs) ∇d ∇sD Zt = θ(B) Θ(Bs) at
  • p,d,q：一般（非季節）部分的 AR 階、差分次數、MA 階
  • P,D,Q：季節部分的 AR 階、差分次數、MA 階
  • s：季節週期（月份 s=12，季度 s=4）
  • ∇s Zt = Zt − Zt-s（季節差分）

範例：航空乘客資料 ARIMA(0,1,1)(0,1,1)₁₂

+

月份航空乘客資料（n=144, 1949~1960），具有上升趨勢與季節性。
先做對數轉換：tf = log(Z)，再識別模式。

proc arima;

proc arima;
  identify var = tf(1);     /* 1 階差分 */
proc arima;
  identify var = tf(1,12);  /* 1 階 + 季節差分 */
  estimate q=(1,12) noconstant printall plot;
  forecast lead=12 out=fcast id=time;
data ex1;
  set fcast;
  z = exp(tf);     /* 轉回原始尺度 */
  fz = exp(forecast);

∴ 模式為 ARIMA(0,1,1)(0,1,1)₁₂（航空模式，Box-Jenkins 的著名範例）

AIC / SBC 越小越好

教授重點筆記

  • Box-Jenkins 三步驟必考：識別(ACF/PACF) → 估計(CLS/MLE) → 診斷(Ljung-Box)。
  • ACF 緩慢遞減 = 非平穩警報！立即差分，直到 ACF 快速衰減。
  • AR → PACF 切斷（決定 p）；MA → ACF 切斷（決定 q）；ARMA → 兩者皆遞減。
  • 平穩條件：AR 多項式的根在單位圓「外」；可逆條件：MA 多項式的根在單位圓「外」。
  • 差分次數 d 的判斷：ACF 不收斂 → d=1；差分後仍不收斂 → d=2（d 很少超過2）。
  • Ljung-Box 不拒絕 H₀ 才代表模式合適（殘差無相關）。
  • 季節資料：先做一般差分(d)消趨勢，再做季節差分(D)消季節性，再看 ACF/PACF 在 s, 2s 的行為。
  • 預測區間隨 l 增大而擴大，因為不確定性累積（ψ 係數越來越多）。
  • 對數轉換常用於具乘法型季節性的資料（如航空客運量），轉換後方差穩定，再建 ARIMA。
  • AIC 傾向選複雜模式（過擬合風險）；SBC 懲罰更重，傾向簡約模式。實務上兩者一起看。
