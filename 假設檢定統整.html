9.1 虛無假設與對立假設 LO 9.1

假設檢定是用樣本資料來挑戰「現狀」（status quo）的統計方法。

H₀ vs Hₐ

H₀（虛無假設）：現狀，包含等號（=, ≤, ≥）
Hₐ（對立假設）：我們想要建立的主張（≠, >, <）

決策規則：
• p-value < α → 拒絕 H₀
• p-value ≥ α → 不拒絕 H₀

注意：我們只說「不拒絕」H₀，永遠不說「接受」H₀。

單尾 vs 雙尾

類型Hₐ 符號p-value 計算
右尾μ > μ₀P(Z ≥ z)
左尾μ < μ₀P(Z ≤ z)
雙尾μ ≠ μ₀2×P(Z ≥ |z|)

要確立的主張放 Hₐ；等號永遠在 H₀。

建立假設例題

例題 9.1｜返校購物平均支出是否為 $606.40？（雙尾）▼

貿易團體預測每戶家庭返校購物支出平均為 $606.40。若預測有誤，需要不同的經濟模型。

H₀: μ = 606.40
Hₐ: μ ≠ 606.40（雙尾，因為「有誤」包含高於或低於）
雙尾檢定

H₀: μ ≤ 10 磅
Hₐ: μ > 10 磅（右尾，主張「超過」放 Hₐ）
右尾檢定

9.2 Type I 與 Type II 誤差 LO 9.2

決策 \ 真相H₀ 為真H₀ 為假
拒絕 H₀Type I 誤差（α）正確決策
不拒絕 H₀正確決策Type II 誤差（β）

重要關係

α = P(Type I 誤差) = 顯著水準
β = P(Type II 誤差)

固定 n：α↓ 則 β↑（互相消長）
同時降低 α 和 β：只能增加 n

例題 9.5｜實體店開設決策

H₀: μ ≤ 500（不開店），Hₐ: μ > 500（開店）

Type I 誤差：μ ≤ 500 但拒絕 H₀
→ 開了店但人流不足，虧損

Type II 誤差：μ > 500 但不拒絕 H₀
→ 沒開店但其實人流充足，錯失機會

9.2 μ 的假設檢定（σ 已知，z 檢定） LO 9.3

檢定統計量

z = (x̄ − μ₀) / (σ/√n)

適用條件：X̄ 服從常態分配（母體常態，或 n ≥ 30）

信賴區間法（雙尾）

x̄ ± zα/2 × σ/√n

拒絕 H₀：若 μ₀ 不在 CI 內
不拒絕 H₀：若 μ₀ 在 CI 內

p-value 法四步驟

1
設定 H₀ 和 Hₐ（確認單/雙尾、包含等號在 H₀）
2
設定顯著水準 α
3
計算 z 統計量與 p-value
4
結論：p < α → 拒絕 H₀；p ≥ α → 不拒絕 H₀

例題

例題 9.6｜返校支出是否異於 $606.40？（雙尾，σ=65，n=30）▼

研究分析師認為支出與 $606.40 不同。n=30，x̄=$622.85，σ=$65，α=0.05

H₀: μ = 606.40，Hₐ: μ ≠ 606.40

z = (622.85 − 606.40) / (65/√30) = 16.45 / 11.863 = 1.3862

雙尾 p-value = 2×P(Z ≥ 1.3862) = 2×(1 − 0.9172) = 0.1657

0.1657 > 0.05 → 不拒絕 H₀
不拒絕 H₀

樣本資料不支持返校支出異於 $606.40 的主張。

例題 9.7｜用 CI 法重做例題 9.6▼

n=30，x̄=622.85，σ=65，α=0.05，z0.025=1.96

CI = 622.85 ± 1.96 × (65/√30)
= 622.85 ± 23.26
= [599.59, 646.11]

μ₀ = 606.40 ∈ [599.59, 646.11] → 不拒絕 H₀

與 p-value 法結論相同。CI 法只適用於雙尾檢定。

9.3 μ 的假設檢定（σ 未知，t 檢定） LO 9.4

檢定統計量

tdf = (x̄ − μ₀) / (s/√n)，df = n − 1

用樣本標準差 s 取代 σ，改用 t 分配。X̄ 需近似常態（母體常態或 n ≥ 30）。

例題

例題 9.8｜學生讀書時間是否低於 24 小時？（左尾）▼

助理院長想確認大學生讀書時間是否低於 1961 年全國平均 24 小時/週。

n=35，x̄=16.3714，s=7.2155，α=0.05

H₀: μ ≥ 24，Hₐ: μ < 24（左尾）

df = 35 − 1 = 34
t₃₄ = (16.3714 − 24) / (7.2155/√35) = −7.6286 / 1.2197 = −6.255

p-value = P(T₃₄ ≤ −6.255) < 0.005（遠小於任何合理 α）

p < 0.05 → 拒絕 H₀
拒絕 H₀

在 5% 顯著水準下，可認為大學生讀書時間顯著低於 24 小時/週。

例題 9.9｜讀書時間是否異於今日全國平均 14 小時？（雙尾）▼

n=35，x̄=16.3714，s=7.2155，μ₀=14，α=0.05

H₀: μ = 14，Hₐ: μ ≠ 14（雙尾）

t₃₄ = (16.3714 − 14) / (7.2155/√35) = 2.3714 / 1.2197 = 1.9444

p-value = 2×P(T₃₄ ≥ 1.9444) = 0.0602

0.0602 > 0.05 → 不拒絕 H₀
不拒絕 H₀

在 5% 顯著水準下，無法認為讀書時間與今日全國平均 14 小時不同。

9.4 母體比例的假設檢定 LO 9.5

檢定統計量

z = (p̄ − p₀) / √(p₀(1−p₀)/n)

p̄ = x/n（樣本比例）
p₀ = 假設母體比例

使用 z 分配。適用條件：np₀ ≥ 5 且 n(1−p₀) ≥ 5。

例題 9.14｜改變生活方式的比例是否低於 40%？（左尾）▼

一雜誌宣稱美國不到 40% 家庭因環保而改變生活方式。調查 n=180，67 戶有改變。α=0.05

H₀: p ≥ 0.40，Hₐ: p < 0.40（左尾）

p̄ = 67/180 = 0.3722
z = (0.3722 − 0.40) / √(0.40×0.60/180)
= −0.0278 / 0.03651 = −0.7613

p-value = P(Z ≤ −0.7613) = 0.2233

0.2233 > 0.05 → 不拒絕 H₀
不拒絕 H₀

無法認為比例顯著低於 40%。

9.5 臨界值法 LO 9.3–9.5

與 p-value 法結論永遠相同，差別在步驟 2 找臨界值、步驟 4 比較統計量落點。

檢定類型z 臨界值拒絕條件
右尾zαz > zα
左尾−zαz < −zα
雙尾±zα/2|z| > zα/2

t 檢定時，將 z 換成 tα,df，方法相同。

例題

例題 9.12｜用臨界值法重做例題 9.6（z 檢定，雙尾）▼
α=0.05，雙尾 → 臨界值 ±z0.025 = ±1.96
決策規則：|z| > 1.96 則拒絕 H₀

z = 1.3862
−1.96 < 1.3862 < 1.96 → 不拒絕 H₀
例題 9.13｜用臨界值法重做例題 9.8（t 檢定，左尾）▼
α=0.05，左尾，df=34 → 臨界值 −t0.05,34 = −1.691
決策規則：t < −1.691 則拒絕 H₀

t = −6.255 < −1.691 → 拒絕 H₀
例題 9.14｜用臨界值法重做比例檢定（左尾）▼
α=0.05，左尾 → 臨界值 −z0.05 = −1.645
決策規則：z < −1.645 則拒絕 H₀

z = −0.7613 > −1.645 → 不拒絕 H₀

9.6 案例研究｜美國所得不平等

地方政客宣稱：① 低收入家庭平均所得低於 2017 年的 $50,000；② 所得低於 $30,000 的比例超過 20%。

樣本：n=36 戶，x̄=$47,278，s=$19,524，低於 $30,000 的家庭 8 戶（p̄=8/36=0.222）

假設統計量p-value結論（α=0.05）
H₀: μ≥50,000
Hₐ: μ<50,000		 t₃₅ = (47278−50000)/(19524/√36) = −0.837 0.204 不拒絕 H₀
H₀: p≤0.20
Hₐ: p>0.20		 z = (0.222−0.20)/√(0.20×0.80/36) = 0.333 0.369 不拒絕 H₀

兩項主張皆無法在統計上獲得支持——政客的說法可能只是基於樣本統計量的直觀觀察，而非嚴謹的假設檢定。

🎓 教授觀點｜期中考重點分析

📌 這章的核心邏輯只有一條

所有假設檢定，不管是 z、t 還是比例，都是同一個流程：

建立假設 → 算統計量 → 算 p-value → 跟 α 比較 → 做結論

差別只在「用哪個統計量」和「查哪個分配」。掌握這條主線，三種檢定全部迎刃而解。

⚠️ 考試最常失分的地方

1. H₀ 和 Hₐ 設錯：等號（=, ≤, ≥）永遠在 H₀，想要建立的主張永遠在 Hₐ。

2. 雙尾忘記乘以 2：雙尾 p-value = 2 × 單尾面積。

3. σ 已知用 z，σ 未知用 t：這題目會直接告訴你，別看漏了。

4. 結論用字：拒絕 H₀ → 支持 Hₐ；不拒絕 H₀ → 不能支持 Hₐ（不是「接受 H₀」）。
