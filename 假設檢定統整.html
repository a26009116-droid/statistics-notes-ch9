9.1 假設檢定基本概念

假設檢定是透過樣本資料，判斷是否有足夠證據推翻某一預設命題的統計程序。

虛無假設 H₀

代表現狀（status quo），必須含等號（=、≤、≥）

對立假設 Hₐ

研究者想確立的新主張，不含等號（≠、>、<）

H₀ 符號Hₐ 符號檢定類型
=≠雙尾檢定
≤>右尾檢定（單尾）
≥<左尾檢定（單尾）
設定 H₀ 與 Hₐ
↓
收集樣本，計算檢定統計量
↓
算出 p 值（或比較臨界值）
↓
p 值 < α
→ 拒絕 H₀
p 值 ≥ α
→ 不拒絕 H₀
重要：結論只能說「拒絕 H₀」或「不拒絕 H₀」，絕對不能說「接受 H₀」。不拒絕 ≠ 為真。
決策 \ 真實情況H₀ 為真H₀ 為假
拒絕 H₀型一錯誤（α）正確決策
不拒絕 H₀正確決策型二錯誤（β）

型一 / 型二 錯誤與檢定力互動圖（拖曳 α 值）

9.2 母體平均數檢定（σ 已知）—— p 值法

σ 已知時，使用 z 統計量，樣本大小需能保證 x̄ 近似常態（n≥30 或母體常態）。

檢定統計量
z = (x̄ − μ₀) / (σ / √n)
x̄：樣本均值　μ₀：假設值　σ：母體標準差　n：樣本數

p 值法四步驟

1
設定假設：確立 H₀ 與 Hₐ（等號放 H₀，研究主張放 Hₐ）
2
設定顯著水準 α：常用 0.01 / 0.05 / 0.10，即型一錯誤機率上限
3
計算 z 值與 p 值：依尾端類型決定 p 值算法（見下表）
4
決策：p 值 < α → 拒絕 H₀；否則不拒絕
對立假設 Hₐp 值計算方式
μ > μ₀（右尾）P(Z ≥ z)
μ < μ₀（左尾）P(Z ≤ z)
μ ≠ μ₀（雙尾）z>0：2×P(Z≥z)；z<0：2×P(Z≤z)

Z 互動式 Z 檢定計算機（σ 已知）

雙尾檢定的信賴區間法
(1−α)×100% CI：x̄ ± zα/2 × σ/√n
μ₀ 落在區間內 → 不拒絕 H₀　｜　μ₀ 落在區間外 → 拒絕 H₀
例題 9.6 — 開學採購支出（p 值法，σ已知，雙尾）▼

研究分析師認為開學每戶花費與 $606.40 不同。n=30，x̄=$622.85，σ=65，α=0.05

H₀: μ = 606.40Hₐ: μ ≠ 606.40
雙尾檢定

z = (622.85 − 606.40) / (65/√30) = 1.3862
p 值 = 2 × P(Z ≥ 1.3862) = 2 × 0.0828 = 0.1657
0.1657 > 0.05 → 不拒絕 H₀

結論：樣本資料不支持均值偏離 $606.40 的主張。
例題 9.7 — 開學採購支出（信賴區間法）▼

同例題 9.6，改用 95% 信賴區間法。z0.025 = 1.96

CI = 622.85 ± 1.96 × (65/√30) = 622.85 ± 23.26 = [599.59, 646.11]
μ₀ = 606.40 落在區間內 → 不拒絕 H₀（結論與 p 值法一致）

例題（退休年齡）— z 值計算示範（右尾）▼

社會學家欲確立平均退休年齡 > 67 歲。σ=9，n=25，x̄=71，α=0.05

H₀: μ ≤ 67Hₐ: μ > 67
右尾檢定

z = (71 − 67) / (9/√25) = 4 / 1.8 = 2.2222
p 值 = P(Z ≥ 2.2222) = 0.0131
0.0131 < 0.05 → 拒絕 H₀

結論：樣本資料支持平均退休年齡大於 67 歲。

9.3 母體平均數檢定（σ 未知）—— t 分配

實務中 σ 幾乎不可知，改用樣本標準差 s，統計量服從 t 分配（自由度 df = n−1）。

t 檢定統計量
tdf = (x̄ − μ₀) / (s / √n)　，　df = n − 1
s：樣本標準差　條件：母體近似常態 或 n ≥ 30

z 分配 vs t 分配

  • z：σ 已知 → 標準常態
  • t：σ 未知 → t 分配（尾端較厚）
  • df 越大，t 越接近 z

使用時機

  • σ 未知（最常見情況）
  • 從樣本計算 s
  • df = n − 1 查 t 表

t 互動式 T 檢定計算機（σ 未知）

例題 9.8 — 學生讀書時數（左尾，拒絕 H₀）▼

系主任主張學生每週讀書 ≥ 24 小時。n=35，x̄=16.3714 hr，s=7.2155，α=0.05

H₀: μ ≥ 24 hrHₐ: μ < 24 hr
左尾檢定

df = 34，t = (16.3714 − 24) / (7.2155/√35) = −6.260
p 值 ≈ P(T ≤ −6.260) ≈ 0.000（極小）
p < 0.05 → 拒絕 H₀

結論：有足夠證據顯示學生平均讀書時數少於 24 小時/週。
例題 9.9 — 學生讀書時數（雙尾，不拒絕 H₀）▼

想確認該校學生讀書時數是否與全國均值 14 小時不同。n=35，x̄=16.37，s=7.22，α=0.05

H₀: μ = 14 hrHₐ: μ ≠ 14 hr
雙尾檢定

t = (16.37 − 14) / (7.22/√35) = −1.9444
p 值 ≈ 0.0600
0.0600 > 0.05 → 不拒絕 H₀

結論：在 5% 顯著水準下，無法認定讀書時數與全國 14 小時不同。

9.4 母體比例的假設檢定

關心類別變數比例 p 時，使用以下 z 統計量（估計量：p̂ = x / n）。

比例檢定統計量
z = (p̂ − p₀) / √( p₀(1−p₀) / n )
使用條件：np₀ > 5　且　n(1−p₀) > 5

p 互動式比例檢定計算機

例題 9.10 — 環保生活方式（左尾，不拒絕）▼

雜誌聲稱少於 40% 家庭因環保改變生活方式。n=180，67 戶確實改變，α=0.05

H₀: p ≥ 0.40Hₐ: p < 0.40
左尾檢定

p̂ = 67/180 = 0.3722
z = (0.3722 − 0.40) / √(0.40×0.60/180) = −0.7607
p 值 = P(Z ≤ −0.7607) = 0.2234
0.2234 > 0.05 → 不拒絕 H₀

結論：無足夠證據支持環保改變比例低於 40%。
例題 9.11 — 大麻合法化支持率（雙尾，α=0.10 拒絕）▼

全國 67% 支持大麻合法化。俄亥俄州 200 人中 122 人支持，α=0.10

H₀: p = 0.67Hₐ: p ≠ 0.67
雙尾檢定

p̂ = 122/200 = 0.61
z = (0.61 − 0.67) / √(0.67×0.33/200) = −1.8046
p 值 = 2×P(Z ≤ −1.8046) = 0.0711
0.0711 < 0.10 → 拒絕 H₀

結論：在 10% 顯著水準下，俄亥俄州支持率與全國 67% 有顯著差異。

例題 9.12 — 開學採購（臨界值法，雙尾，z）▼

同例 9.6。n=30，x̄=622.85，σ=65，α=0.05

H₀: μ = 606.40Hₐ: μ ≠ 606.40

臨界值：±z0.025 = ±1.96，拒絕域：z < −1.96 或 z > 1.96
z = 1.3862　→　−1.96 < 1.3862 < 1.96　→　不拒絕 H₀（結論同 p 值法）

例題 9.13 — 讀書時數（臨界值法，左尾，t）▼

同例 9.8。n=35，x̄=16.3714，s=7.2155，α=0.05

H₀: μ ≥ 24Hₐ: μ < 24

df=34，臨界值：−t0.05,34 = −1.691，拒絕域：t < −1.691
t = −6.260　→　−6.260 < −1.691　→　拒絕 H₀（結論同 p 值法）

例題 9.14 — 環保比例（臨界值法，左尾，z）▼

同例 9.10。n=180，p₀=0.40，α=0.05

H₀: p ≥ 0.40Hₐ: p < 0.40

臨界值：−z0.05 = −1.645，拒絕域：z < −1.645
z = −0.7613　→　−0.7613 > −1.645　→　不拒絕 H₀（結論同 p 值法）

9.6 數字的溝通 — 案例研究（所得不平等）

某地方政治人物宣稱：
① 低收入家庭平均薪資已低於 2017 年均值 $50,000
② 超過 20% 家庭收入低於 $30,000

調查 n=36 戶低收入家庭：x̄=$47,278、s=$19,524；8 戶（22.2%）低於 $30,000

假設（Hₐ 為政治人物的主張） 檢定統計量 p 值 結論（α=0.05）
H₀: μ ≥ 50,000
Hₐ: μ < 50,000		 t = (47278−50000)/(19524/√36) = −0.837 0.204 不拒絕 H₀
H₀: p ≤ 0.20
Hₐ: p > 0.20		 z = (0.222−0.20)/√(0.20×0.80/36) = 0.333 0.369 不拒絕 H₀
統計結論：兩項聲稱均無法以樣本資料在統計上成立。政治人物的說法可能基於表面數字觀察，而非嚴謹的假設檢定。
本案例啟示：p 值不僅是數字，更是溝通工具。在公共政策討論中，提供檢定統計量與 p 值可增強論述的說服力與可信度。

章節公式摘要

Z 檢定（μ，σ已知）
z = (x̄ − μ₀) / (σ/√n)
∼ 標準常態 Z

t = (x̄ − μ₀) / (s/√n)，df = n−1
∼ t 分配

z = (p̂ − p₀) / √(p₀(1−p₀)/n)
np₀>5 且 n(1−p₀)>5

x̄ ± 1.96 × σ/√n
雙尾用於判斷是否拒絕
